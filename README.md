
# 🚀 麦麦插件整合仓库

## 介绍
本仓库仅用于整合麦麦插件 v0.1。

> ⚠️ 当前暂未适配 v0.2 版本插件。

## 主旨
本仓库旨在为用户提供一个集中、便捷的插件下载与使用平台。

## 贡献指南

欢迎各位开发者提交您的插件！请确保提交的内容符合以下要求：

📌 **请将插件zip格式压缩包 PR 提交至 `plugin` 分支！**

请在 插件ZIP 文件中包含以下内容：

1. 插件文件夹（需包含插件的 `.py` 文件及 `__init__.py`）
2. 教程文档（推荐使用 Markdown 编写，说明插件功能及使用方式）
