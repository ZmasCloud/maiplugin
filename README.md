
# 🚀 麦麦插件整合仓库

## 介绍
本仓库仅用于整合麦麦插件 v0.1。

> ⚠️ 当前暂未适配 v0.2 版本插件。

## 主旨
本仓库旨在为用户提供一个集中、便捷的插件下载与使用平台。

## 贡献指南

欢迎各位开发者提交您的插件！请确保提交的内容符合以下要求：

📌 **请将插件about.md文件 PR 提交至 `plugin` 分支！**

请在 about.md 文件中编写一下内容内容：

> | 插件名称 | 仓库地址 | 插件作者 |
> |----------|----------|----------|
> | 内容     | [内容](url) | [内容](url) |
```markdown
 |----------|----------|----------|
 | 插件名称     | [gitee/github](仓库url) | [作者主页](主页url) |
```

>>请不要修改markdown格式！
